Stophy gives developers a simple, reliable HTTP API to extract data from YouTube videos. Fetch transcripts in any available language, retrieve rich video metadata, or paginate through comments — all protected by API key auth and backed by intelligent Redis caching to minimize latency and credit usage.

## How it works

<Steps>
  <Step title="Get your API key">
    Create an account and generate an API key from your dashboard. Each key is tied to your credit balance.
  </Step>
  <Step title="Make a request">
    Call any endpoint with your API key in the `x-api-key` header and a YouTube video URL in the request body.
  </Step>
  <Step title="Receive structured data">
    Get back clean, structured JSON — transcripts as plain text and timed segments, video metadata, or paginated comments.
  </Step>
  <Step title="Monitor your usage">
    Each successful request deducts 1 credit. Cached responses still cost 1 credit but return instantly.
  </Step>
</Steps>

## Key features

- **Multi-language transcripts** — request transcripts in any language code (e.g., `en`, `es`, `fr`)
- **Smart caching** — Redis-backed cache means repeat requests return in milliseconds
- **Atomic credit deductions** — race-condition-safe credit management ensures you're never overcharged
- **Activity logging** — every request is logged with status, response time, and credits used
- **Simple auth** — pass your API key in the `x-api-key` or `api-x-key` header
