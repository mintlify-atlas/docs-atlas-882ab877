<Steps>
  <Step title="Get your API key">
    Create an account and generate an API key from your dashboard. Your key is a secret string you pass in every request header.

    Stophy accepts your API key in either of two headers — use whichever you prefer:

    - `x-api-key`
    - `api-x-key`

    Both are equivalent. The server checks for `api-x-key` first, then falls back to `x-api-key`.

    <Warning>
      Keep your API key secret. Do not commit it to source control or expose it in client-side code.
    </Warning>
  </Step>

  <Step title="Fetch a transcript">
    Call `POST /v1/video/transcript` with your API key header and a JSON body containing the YouTube video URL.

    <CodeGroup>

    ```bash curl
    curl -X POST https://your-stophy-host/v1/video/transcript \
      -H "Content-Type: application/json" \
      -H "x-api-key: YOUR_API_KEY" \
      -d '{"videoUrl": "https://www.youtube.com/watch?v=dQw4w9WgXcQ", "languageCode": "en"}'
    ```

    ```javascript JavaScript
    const response = await fetch('https://your-stophy-host/v1/video/transcript', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
        'x-api-key': 'YOUR_API_KEY',
      },
      body: JSON.stringify({
        videoUrl: 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
        languageCode: 'en',
      }),
    });

    const data = await response.json();
    console.log(data);
    ```

    ```python Python
    import requests

    response = requests.post(
        'https://your-stophy-host/v1/video/transcript',
        headers={'x-api-key': 'YOUR_API_KEY'},
        json={
            'videoUrl': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
            'languageCode': 'en',
        },
    )

    print(response.json())
    ```

    </CodeGroup>

    The `languageCode` parameter is optional and defaults to `"en"` if omitted.
  </Step>

  <Step title="Read the response">
    A successful response returns HTTP 200 with the following structure:

    ```json
    {
      "statusCode": 200,
      "cacheStatus": "miss",
      "creditUsed": 1,
      "remainingCredits": 99,
      "transcript": {
        "plainText": "Never gonna give you up, never gonna let you down...",
        "transcript": [
          {
            "text": "Never gonna give you up",
            "start": 18.24,
            "duration": 1.44
          },
          {
            "text": "never gonna let you down",
            "start": 19.68,
            "duration": 1.44
          }
        ]
      }
    }
    ```

    - `cacheStatus` — `"hit"` if returned from cache, `"miss"` if freshly fetched
    - `creditUsed` — always `1` for a successful request
    - `remainingCredits` — your credit balance after this request
    - `transcript.plainText` — the full transcript as a single string
    - `transcript.transcript` — array of timed segments, each with `text`, `start` (seconds), and `duration` (seconds)

    <Tip>
      If you call the same endpoint again with the same video URL and language code, the response comes from cache. `cacheStatus` will be `"hit"` and the response will be faster — but still costs 1 credit.
    </Tip>
  </Step>
</Steps>

## Other endpoints

Once you have transcripts working, try the other two endpoints.

### Fetch video details

`POST /v1/video/details` returns metadata for a video.

<CodeGroup>

```bash curl
curl -X POST https://your-stophy-host/v1/video/details \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"videoUrl": "https://www.youtube.com/watch?v=dQw4w9WgXcQ"}'
```

```javascript JavaScript
const response = await fetch('https://your-stophy-host/v1/video/details', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY',
  },
  body: JSON.stringify({
    videoUrl: 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://your-stophy-host/v1/video/details',
    headers={'x-api-key': 'YOUR_API_KEY'},
    json={'videoUrl': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ'},
)

print(response.json())
```

### Fetch video comments

`POST /v1/video/comments` returns paginated comments. Use `sortBy` (`"top"` or `"newest"`) to control ordering, and pass `continuationToken` to fetch subsequent pages.

<CodeGroup>

```bash curl
curl -X POST https://your-stophy-host/v1/video/comments \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{"videoUrl": "https://www.youtube.com/watch?v=dQw4w9WgXcQ", "sortBy": "top"}'
```

```javascript JavaScript
const response = await fetch('https://your-stophy-host/v1/video/comments', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_API_KEY',
  },
  body: JSON.stringify({
    videoUrl: 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
    sortBy: 'top',
  }),
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://your-stophy-host/v1/video/comments',
    headers={'x-api-key': 'YOUR_API_KEY'},
    json={
        'videoUrl': 'https://www.youtube.com/watch?v=dQw4w9WgXcQ',
        'sortBy': 'top',
    },
)

print(response.json())
```

</CodeGroup>

```json
{
  "statusCode": 200,
  "cacheStatus": "miss",
  "creditUsed": 1,
  "remainingCredits": 97,
  "comments": [
    {
      "id": "UgxABC123",
      "content": "This song never gets old.",
      "publishDate": "2 years ago",
      "likeCount": 45200,
      "isPinned": false,
      "isAuthorChannelOwner": false,
      "replyCount": 312,
      "url": "https://www.youtube.com/watch?v=dQw4w9WgXcQ&lc=UgxABC123",
      "author": { "id": "UCxxxxxx", "name": "Some User", "handle": "@someuser" }
    }
  ],
  "filters": { "sortBy": "top" },
  "pagination": {
    "continuationTokenUsed": null,
    "nextContinuationToken": "EqYCCg8..."
  }
}
```

Pass `nextContinuationToken` as `continuationToken` in your next request to fetch the following page of comments.
